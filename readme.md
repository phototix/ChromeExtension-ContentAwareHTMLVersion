This ChromeExtension main feature is to allow user to use OpenAI to ask contents from the HTML of the active page.

- Popup to let user key in API Key, Model selection & system prompt, save in localStorage.
- Popup allow user to create & login via masterAuthAPI to upload/download the config from/to localStorage.
- Have a DevTool tab dedicated interface to let user to chat with AI, the AI will read the active tab website's HTML & contents and response to user request.
- Preserve user chat history in localStorage.
